### How to use it:

1. download and save the source code.

2. $ make

3. $ ngrams --type=word --n=3 --in= sample.txt

     or 

   $ ngrams --type=character -n=3 --in= sample.txt 

     or

   Byte ngrams, e.g., getting ngrams from binary file.

   $ ngrams --type=byte -n=3 --in= sample.txt 


That's it.
